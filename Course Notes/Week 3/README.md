## Week 3

The five lessons of this week complete your study of basic Julia. At the end of it, you should have the basics of functions, multiple dispatch, branching, and iteration. These should give you an excellent starting platform for writing Julia code.


#### Of course, this is in addition to being able to write valid expressions in Julia, write your code in plain text files and run them, and debugging your code.
